# Samur.AI

## 📒 Descrição
Utilizei a intelgência artificial para crir um viedo de um avatar baseado em um samurai japonês.

## 🤖 Tecnologias Utilizadas
Leonardo.AI

## 🧐 Processo de Criação
1º) Utilizei o módulo de Image Creation e solicitei passei o comando: "Crie uma imagem de um samurai japonês dos anos 1500. Apresente o samurai de corpo inteiro, usandod uma armadura tradicional da época e em alguma posição de luta";
2º) Selecionei duas das quatro imagens geradas e usei o módulo Motion para dar movimento à elas.
