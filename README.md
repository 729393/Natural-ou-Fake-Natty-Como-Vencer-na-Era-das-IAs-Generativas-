# Natural ou Fake Natty? Como Vencer na Era das IAs Generativas

```markdown
# Gato triste em Paris

## 📒 Descrição
A história de um gato triste que encontrou a ajuda de um amigo para ficar feliz

## 🤖 Tecnologias Utilizadas
Microsoft Designer
Microsoft Clipchamp

## 🧐 Processo de Criação
Utilizei o Microsoft Designer para gerar as imagens com prompts em inglês pois em português a resposta ainda não é tão satisfatória.
Depois fiz um video unindo todas as imagens criadas no Microsoft Clipchamp, narrei a história através da ferramenta de IA que transforma texto em audio da própria ferramenta.

## 🚀 Resultados
Um video curto que conta a história de um gato triste recebendo a ajuda de um amigo.
Link do **[video](https://drive.google.com/file/d/1sKo3jdJZk_ACiT8ibVJlEtXDD_ZnZcOr/view?usp=sharing)**


## 💭 Reflexão (Opcional)
O desafio foi bem interessante porque permitiu que eu explorasse uma nova ferramenta que foi o Microsoft Designer.
```
