# Ebook: Java com Inteligência Artificial: Aprenda Back-end de Forma Prática e Moderna ;)

## 📒 Descrição
Este e-book de 115 páginas explora de forma atraente e concisa como aprender Java e utilizar o poder das IAs generativas para auxiliar na explicação de conteúdo, resolução e criação de exercícios para lhe ajudar e ser tornar um desenvolvedor de destaque.

## 🤖 Tecnologias Utilizadas
- IA Generativa **[ChatGPT](https://chat.openai.com)** para roteirização;
- IA Generativa **[Copilot AI](https://copilot.microsoft.com/images/create)** para criar a arte visual do e-book;
- Ferramenta **Textmaker** para edição.

## 🧐 Processo de Criação
Utilizei o "ChatGPT" para revisar minha pesquisa sobre Java com foco em back-end, estruturar o conteúdo e refinar minhas ideias. Por outro lado, o "Copilot AI" foi usado para gerar imagens exclusivas para o nosso e-book. Para concluir, a montagem final foi feita no Textmaker, por ser uma ferramenta open-source para edição de texto em LaTeX e gerador de PDF.
